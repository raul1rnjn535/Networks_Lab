Steps to execute the udpServer:

1. Compile udpServer.c
   gcc udpServer.c -o udpServer

2. Execute the object file using PORT_NO as arguement
   ./udpServer PORT_NO

Steps to execute the udpClient:

1. Compile udpClient.c
   gcc udpClient.c -o udpClient

2. Execute the object file using PORT_NO and server_IP_address as arguements
   ./udpClient PORT_NO server_IP_address
