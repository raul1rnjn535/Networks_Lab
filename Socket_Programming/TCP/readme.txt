Steps to execute the tcpServer:

1. Compile tcpServer.c
   gcc tcpServer.c -o tcpServer

2. Execute the object file using PORT_NO as arguement
   ./tcpServer PORT_NO

Steps to execute the tcpClient:

1. Compile tcpClient.c
   gcc tcpClient.c -o tcpClient

2. Execute the object file using PORT_NO and server_IP_address as arguements
   ./tcpClient PORT_NO server_IP_address
